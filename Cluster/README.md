[KIND 설치](https://kind.sigs.k8s.io/docs/user/quick-start/#installation)
(도커가 미리 설치 되어 있어야 합니다)

```
brew install kind
```
KIND 로 클러스터를 만들어 봅시다.
```
kind create cluster --name k8s101-kind --config kind-config.yaml
```
